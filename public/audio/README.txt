Cold Case Files — audio assets
==============================

Drop these files here (exact names). The app works without them (it just
stays silent); the moment a file exists, that sound plays. Sound is OFF by
default and toggled from the header speaker icon (choice is remembered).

Required files
--------------
ambient.mp3   Looping noir atmosphere bed (rain / low drone / distant city).
              ~30–60s seamless loop, quiet. Plays across the app when sound is on.

examine.mp3   Short paper/file shuffle. Fires when an exhibit is examined.
pin.mp3       Thumbtack push. Fires when a lead is pinned on the board.
cut.mp3       Scissor snip. Fires when a board thread is cut.
correct.mp3   Positive sting. Fires on a correct chapter puzzle answer.
wrong.mp3     Negative buzz/thud. Fires on a wrong answer.
unlock.mp3    Dramatic sting. Fires when a chapter cinematic plays.

Notes
-----
- Keep them small: ambient < ~1 MB, each SFX < ~80 KB. MP3 (or OGG/M4A if you
  also update the paths) is fine.
- Use sounds you have the right to use (royalty-free / CC0 / your own).
